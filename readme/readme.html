Installation: Famous 2-minute install

  1. Login to your server via SSH Access.
  2. Upload our PassUp package into your server webroot directory(Ex. /home/username/public_html).
  3. Unzip our PassUp script and make sure you have already installed Node.js, mongoDB and Apache webserver on your server machine.
  4. Go to our PassUp directory and Run the deploy.sh file by using this command (sh deploy.sh)
  5. After running the Shell script it will aks some details from you on terminal, Like mongoDB name, Username, Pssword, Site name, , Site url, Site admin username and password, Port number.
    1. If you have some error on the above steps, Please delete the.env file and replace it with .env.example file from .env
  6. It will take some time to setup passup after this process completed.

System Requirements

  • Node.js version v10.*.
  • mongoDB version 3.0.
  • The Apache Web Server version 2.4.
  • The PM2 Node module.
